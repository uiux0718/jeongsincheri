🍒 정신체리 (Jeongsin-Cherry)
따뜻한 위로와 냉철한 뇌과학의 만남: AI 심리 코칭 서비스

"지친 일상, 전두엽의 피로를 씻어내는 1분간의 쉼표"

🔍 프로젝트 개요
'정신체리'는 과부하된 일상을 살아가는 현대인을 위해 감성적 지지(Empathy)와 인지 행동 치료(CBT)를 결합한 하이브리드형 AI 심리 코칭 서비스입니다. 단순한 대화를 넘어, 사용자의 뇌 피로도를 시각화하고 맞춤형 미션을 제안하는 '사용자 중심의 멘탈 헬스케어 솔루션'을 지향합니다.

🎨 UX/UI Design Philosophy (디자이너 핵심 역량 강조)
본 프로젝트는 '불안을 다루는 서비스는 인터페이스부터 차분해야 한다'는 원칙 아래 설계되었습니다.

Emotional UI/UX: 감정을 수용하는 단계에 맞춰 뉴트로 감성의 따뜻한 컬러 팔레트와 부드러운 인터랙션을 적용했습니다.

Cognitive Load Reduction: 학술적 용어를 배제하고 친근한 어투를 사용하여 사용자가 대화 과정에서 느끼는 인지적 피로도를 최소화했습니다.

Gamification: 성취감이 필요한 행동 요법 단계에 콘페티(Confetti) 이펙트를 도입하여 긍정적인 행동 강화와 서비스 몰입도를 높였습니다.

Data Visualization: 복잡한 심리 리포트를 Recharts를 활용해 직관적인 차트로 시각화하여 사용자가 자신의 상태를 한눈에 파악할 수 있도록 설계했습니다.

🚀 주요 기능 (Key Features)
전두엽 피로도 진단: 슬라이더 인터페이스를 통해 사용자의 감정 상태를 정량적으로 측정합니다.

3단계 CBT 프로세스: 정서적 수용(Emotional) → 이성적 필터링(Rational) → 행동 처방(Actionable)으로 이어지는 사용자 여정(User Journey)을 설계했습니다.

개인화 필터링: 사용자가 기피하는 표현을 원천 차단하여 사용자 중심의 안전한 상담 환경을 구현했습니다.

연속성 보존: 대화 문맥을 파악하여 기억에 남는 개인화된 경험을 제공합니다.

리포트 시각화: 상담 기록을 PDF로 저장하여 사용자가 스스로의 마음 건강 변화를 기록/관리할 수 있습니다.

🛠 Tech Stack
Frontend: React (v19), Tailwind CSS (v4), Framer Motion, Recharts, Lucide React, Canvas Confetti

Backend & AI: Express/Node.js, Google GenAI SDK (Gemini 3.5 Flash), Firebase Firestore, TypeScript
