# 🍒 정신체리 (Jeongsin-Cherry)
> **따뜻한 위로와 냉철한 뇌과학의 만남, AI 심리 코칭 및 상담 서비스**

정신체리는 과부하된 일상을 살아가는 현대인들을 위해 따뜻한 감성적 지지와 이성적 인지 행동 치료(CBT) 요법을 결합한 **하이브리드형 AI 심리 코칭 서비스**입니다.  
학술적인 딱딱한 표현을 걷어내고 친근한 선배처럼 다가오는 AI 카운셀러 **'정신체리'**와 함께, 지친 머릿속 전두엽의 피로를 씻어내고 평온한 일상의 궤도로 복귀해 보세요.

---

## 🎨 서비스 핵심 가치
1. **공감과 이성의 균형 (Emotion 50% + Rational 50%)**
   - 단순한 영혼 없는 위로나 뻔한 격려에 그치지 않습니다. 감정을 온전히 수용하면서도, 불안이 만들어낸 가짜 생각과 객관적인 사실을 투명하게 분리하도록 도와줍니다.
2. **실천 중심의 초소형 행동 요법 (Micro-Actions)**
   - 자리에 누워만 있고 싶은 무기력한 날에도 즉시 행할 수 있는 1분 미만짜리 초소형 행동(예: "창문 열고 찬 바람 5초 쐬기")을 처방하여 뇌에 활력 신호를 전달합니다.
3. **원활하고 끊김 없는 사용자 경험 (Graceful Hybrid Fallback)**
   - API 제한이나 네트워크 장애 상황에서도 멈추지 않는 지능형 로컬 시나리오 엔진을 탑재하여, 언제 어떤 상황에서도 유저의 곁을 든든하게 지켜줍니다.

## 🚀 주요 기능 (Functional Requirements)

### 1. 전두엽 피로도 및 정서 진단
* **정량적 상태 측정**: 서비스 진입 시, 오늘 하루 겪은 전두엽 피로도와 정서 상태를 직관적인 1점~10점 슬라이더로 진단합니다.
* **실시간 동기화율 추적**: 대화 진행 상태 및 진단 점수를 바탕으로 유저의 '뇌-정서 동기화율'을 실시간 대시보드에 반영합니다.

### 2. 3단계 인지 치료(CBT) 프로세스
대화가 심화됨에 따라 자연스럽게 대화의 깊이와 성격이 유기적으로 진화합니다.
* **1단계: Emotional (정서적 수용)** - 슬픔, 짜증, 불안 등 지친 유저의 마음을 100% 내 편이 되어 따뜻하게 품어주고 경청하는 단계.
* **2단계: Rational (이성적 필터링)** - 유저를 짓누르는 걱정이 실재하는 '사실(Fact)'인지, 불안이 키워낸 '착각/의견(Opinion)'인지 선을 그어 파악하도록 이끄는 단계.
* **3단계: Actionable (행동 처방)** - 지금 앉은 자리에서 가장 쉽게 도전해 볼 수 있는 맞춤형 초소형 미션을 제안하는 단계.

### 3. 맞춤형 솔루션 카드 및 피드백 처방
* **실시간 카드 생성**: 상담 상태와 입력 키워드에 최적화된 **Insight(인사이트)**, **Fact Check(팩트 체크)**, **Mission(미션)** 카드가 처방됩니다.
* **성취감 극대화**: 추천 미션을 실행하고 '완료' 버튼을 누르면 화면 가득 축하 콘페티(Confetti) 이펙트가 펼쳐지며 긍정적인 행동 강화(Positive Reinforcement)를 유도합니다.

### 4. 극비 필터링 지침 & 개인화 시스템
* **기피 표현 필터링 (Hated Words Filter)**: 유저가 사전에 등록한 '제일 듣기 싫어하는 말'(예: 꼰대식 훈계, "노력해라" 등)을 분석하여, AI 상담 답변 내에서 유사한 표현이나 뉘앙스를 원천적으로 회피합니다.
* **상황 맞춤형 페르소나**: 유저의 직업(스타트업, 프리랜서, 학생 등)과 처한 스트레스 맥락을 백그라운드로 인지하여 고도로 정제된 맞춤형 공감을 건넵니다.
* **일상적인 따뜻한 말투**: 어려운 학술 전문 용어(예: 신경가소성, 편도체 등)는 전혀 쓰지 않으며, 친근하고 따뜻한 어투를 엄격하게 고수합니다.

### 5. AI 요약 및 대화 연속성 보존
* **문맥 흐름 파악**: 이전 대화들을 백그라운드에서 실시간 분석 및 요약하여, 유저가 다시 말을 걸었을 때 "지난번에 말씀하셨던 일은 좀 어떠셨어요?"와 같이 감동적인 연속성 있는 대화를 이어갑니다.

### 6. 정서 트래킹 대시보드 & 리포트 PDF 저장
* **시각화 차트**: Recharts를 활용하여 요일별/세션별 감정 상태 및 뇌 피로도 트렌드를 꺾은선 및 영역형 그래프로 시각화합니다.
* **분석 보고서 파일화**: 상담 기록과 최종 처방 카드를 포함한 종합 마음 건강 리포트를 PDF 문서로 즉시 인쇄 및 저장할 수 있습니다.

## 🛠️ Tech Stack

### Frontend
- **React (v19)**: 고성능 컴포넌트 아키텍처 및 상태 관리
- **Tailwind CSS (v4)**: 모던하고 감성적인 다크/소프트 뉴트로 비주얼 테마 구현
- **Framer Motion (`motion`)**: 부드럽고 생동감 넘치는 화면 전환 및 컴포넌트 모션
- **Recharts**: 정서 및 뇌 피로 트렌드 시각화 차트
- **Lucide React**: 직관적이고 미려한 벡터 아이콘 팩
- **Canvas Confetti**: 미션 성취 시 게이미피케이션 효과 구현

### Backend & AI Engine
- **Express / Node.js**: 견고하고 빠른 REST API 서버 구축 및 정적 애셋 서빙
- **Google GenAI SDK (`@google/genai`)**: 최신 Gemini 3.5 Flash 모델 기반 정서 분석 코칭 적용
- **Tsx & Esbuild**: 빠르고 가벼운 TypeScript 실시간 트랜스파일 및 프로덕션 번들링
- **Firebase Firestore**: 상담 이력, 사용자 맞춤 설정 및 세션 기록 데이터 영속화

## 🚀 로컬 실행 방법 (How to Run)

### 1. Prerequisites (사전 준비)
* 시스템에 [Node.js](https://nodejs.org/) (LTS 버전 권장, v18 이상)가 설치되어 있어야 합니다.

### 2. 패키지 설치
프로젝트 루트 경로에서 아래 명령어를 실행하여 필요한 의존성을 설치합니다.
```bash
npm install
```

### 3. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env.local` 또는 `.env` 파일을 생성하고 아래와 같이 Gemini API 키를 입력합니다.
```env
# .env.local
GEMINI_API_KEY=your_actual_gemini_api_key_here
```
> ※ `GEMINI_API_KEY`가 생략되거나 비어 있을 경우, 자동으로 내장된 **지능형 로컬 위로 시나리오 엔진**이 구동되어 오프라인 모드로 안전하게 작동합니다.

### 4. 로컬 개발 서버 실행
```bash
npm run dev
```
실행이 완료되면 기본적으로 `http://localhost:3000` 포트에서 백엔드와 프론트엔드가 실시간 핫 리로딩(Hot Reloading) 상태로 구동됩니다. 브라우저로 접속해 정신체리를 만나보세요!

## 📊 빌드 및 배포 방법 (Build & Deployment)

### 프로덕션 빌드 생성
```bash
npm run build
```
Vite 클라이언트 에셋 컴파일 및 `esbuild`를 통해 TypeScript 백엔드 서버를 단일 번들 파일인 `dist/server.cjs`로 완전하게 컴파일합니다.

### 프로덕션 서버 구동
```bash
npm run start
```
컴파일된 안전한 ESM/CJS 빌드 버전으로 단독 웹서버가 기동됩니다.
